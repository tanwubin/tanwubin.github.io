import numpy as np

class Variable:
    def __init__(self, data):
        self.data = data

data = np.array(1.0)
x = Variable(data)
print(x.data)

x.data = np.array(2.0)
print(x.data)

The Class Variable likes a box to store the data, while the data is not only a number or called scalar,it can be a vector or an array also.
